# The Rank-Size Ruler
A desktop GUI for analyzing the rank-size rule on population datasets.
Use the demo buttons or the example files in /!demo to try out the features.

# Libraries
- pyqtgraph
- PySide6
- numpy

# Data Sources
- Demo (Americas and Asia): ["Zipf's Law, or the Rank-Size Distribution" by Charles van Marrewijk from Utrecht University](http://www2.econ.uu.nl/users/marrewijk/international/zipf.htm)*
- Default World Cities: [UN Department of Economic and Social Affairs, Population Division](https://population.un.org/wup/assets/Download/Cities/WUP2025-F21-DEGURBA-Cities_Pop.xlsx)
- Default Development Indicators: [UN Human Development Reports](https://hdr.undp.org/sites/default/files/2025_HDR/HDR25_Composite_indices_complete_time_series.csv)
- ISO3 Codes: [Country Codes Alpha-2 & Alpha-3](https://www.iban.com/country-codes)
- US State Postal Codes: [Two-Letter State and Territory Abbreviations](https://www.faa.gov/air_traffic/publications/atpubs/cnt_html/appendix_a.html)
